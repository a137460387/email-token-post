# Outlook邮箱令牌取件系统 - 实现方案

## 项目概述
构建一个类似 mail.xiongmaodianjing.top/token 的Web应用，用于导入Outlook邮箱令牌、收邮件、查看最新邮件和历史邮件，支持多账号管理。

## 技术栈
- **后端**: Python + Flask
- **前端**: 原生 HTML/CSS/JavaScript
- **存储**: JSON文件（accounts.json）
- **邮件API**: Microsoft Graph API + OAuth2 Refresh Token

## 项目结构
```
email-token-post/
├── app.py                  # Flask主应用
├── accounts.json           # 账号数据存储

```

## 后端 API 设计

### 账号管理
| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/api/accounts` | 获取所有账号列表 |
| POST | `/api/accounts/import` | 批量导入账号（文本格式） |
| DELETE | `/api/accounts/<id>` | 删除指定账号 |
| DELETE | `/api/accounts/batch` | 批量删除账号 |

### 邮件操作
| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/api/emails/<account_id>` | 获取账号的邮件列表 |
| GET | `/api/emails/<account_id>/<message_id>` | 获取指定邮件详情 |
| POST | `/api/emails/<account_id>/refresh` | 刷新该账号的access_token |

## 核心功能实现

### 1. 账号导入
- 支持文本格式：`email----password----client_id----refresh_token`
- 支持TXT文件上传
- 支持手动输入
- 导入后存储到 accounts.json

### 2. 邮件获取（Microsoft Graph API）
```
刷新Token流程：
1. POST https://login.microsoftonline.com/common/oauth2/v2.0/token
   - grant_type=refresh_token
   - client_id=<client_id>
   - refresh_token=<refresh_token>
   - scope=https://graph.microsoft.com/Mail.Read
   
2. 用获取的access_token调用Graph API：
   GET https://graph.microsoft.com/v1.0/me/messages
   - Authorization: Bearer <access_token>
   - $top=20 (最新20封)
   - $orderby=receivedDateTime desc
```

### 3. 前端界面设计
参考原网站布局：
- **顶部标题栏**: 系统名称 + 导入按钮
- **左侧**: 账号列表（可勾选、分组）
- **右侧**: 邮件列表 / 邮件详情
- **弹窗**: 导入账号、查看邮件详情

### 4. 多账号管理
- 支持全选/反选
- 批量删除
- 每个账号独立刷新token
- 显示账号状态（有效/失效）
